# 초근 누구 v0.2.1

NEIS 초과근무 확인목록을 캘린더와 표로 정리하고, BC카드 이용내역 엑셀을 선택으로 불러와 날짜별 식사 후보 결제와 1인 9,000원 기준을 함께 확인하는 도구입니다.

## v0.2.1 기능

- NEIS 초과근무 최종자료 엑셀 업로드
- 날짜별 초과근무자 캘린더 표시
- 날짜 클릭 시 초과근무자 상세 보기
- 실제 00:00 제외/포함 토글
- 승인상태 필터, 성명 검색
- 매식 가능액 자동 계산: 실제 초근 건수 × 9,000원
- BC카드 이용내역 엑셀 또는 엑셀이 들어있는 ZIP 업로드
- 카드 승인일자 기준 날짜별 식사 후보 결제 표시
- 카드 식사 후보금액과 초근자 수를 비교해 초근없음/필요인원/금액가능 표시
- 카드 전체 보기 토글
- 초과근무 표, 카드 표, 복사용 텍스트 제공

## 카드내역 기준

BC카드 이용내역 엑셀의 다음 열을 자동 인식합니다.

- 접수일자/(승인일자)
- 카드번호
- 승인번호
- 가맹점명/국가명(도시명)
- 청구금액
- 매출종류
- 비고

날짜 비교는 괄호 안의 승인일자를 기준으로 합니다.
PDF는 화면 출력용 성격이 강하고 금액 열 구조가 엑셀보다 불안정하여 v0.2.1에서는 엑셀 또는 ZIP 사용을 권장합니다.

## 주의

식사 후보는 가맹점명과 금액 패턴으로 자동 추정합니다. 최종 특근매식비 인정 여부는 원인행위, 품의, 실제 사용 목적 등 업무담당자가 확인해야 합니다.
