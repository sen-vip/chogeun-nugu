# 초근 누구 🕰️

NEIS 초과근무자료를 캘린더와 표로 정리해, 특근매식비 검토를 쉽게 도와주는 확인 보조 도구입니다.

초과근무자료만 넣어도 사용할 수 있고, 필요하면 BC카드 승인내역 또는 이용내역 엑셀을 추가해 날짜별 식사후보 금액까지 함께 확인할 수 있습니다.

---

## 이런 도구예요 🧾

초근 누구는 초과근무 raw data를 실무자가 보기 좋은 형태로 바꿔주는 도구입니다.

- 날짜별 초과근무자 확인
- 개인별 초과근무 내역 확인
- 실제 초과근무 00:00 제외/포함
- 특근매식비 검토용 매식 가능액 계산
- BC카드 승인내역/이용내역 선택 업로드
- ZIP 안 엑셀 파일 지원
- 승인내역 취소 건 순액 정리
- 식사후보 금액과 초근자 수 비교
- 기준이내, 초과의심, 근거없음 상태 표시
- 이름 가리기 기능
- 표 보기와 복사용 텍스트 제공

이 도구는 최종 판단 도구가 아니라, 자료를 보기 쉽게 정리하는 확인 보조 도구입니다.

---

## 사용 방법 📌

1. NEIS 초과근무자료를 업로드합니다.
2. 날짜별 초과근무자를 캘린더에서 확인합니다.
3. 필요 시 BC카드 승인내역 또는 이용내역을 추가합니다.
4. 식사후보 금액과 초근자 수를 함께 확인합니다.
5. 필요한 경우 날짜별 상세 내역이나 표를 복사해 검토에 활용합니다.

초과근무자료만 넣어도 사용할 수 있습니다.  
BC카드 자료는 선택 기능입니다.

---

## 자료 받는 법 📂

### NEIS 초과근무자료

나이스 접속 → 초과근무 확인목록 → 해당 월 조회 → 엑셀 다운로드

기관별 메뉴명이나 권한에 따라 경로가 다를 수 있습니다.

### BC카드 자료

BC카드 홈페이지 → 승인내역조회 또는 이용내역조회 → 해당 월 조회 → 엑셀 다운로드

초근 누구는 다음 파일을 지원합니다.

- BC카드 승인내역조회 엑셀
- BC카드 이용내역조회 엑셀
- 엑셀이 들어있는 ZIP 파일

PDF는 사람이 보기에는 좋지만, 자동 분석용으로는 엑셀보다 불안정할 수 있습니다.

---

## 주요 기능 🔍

### 초과근무 캘린더

NEIS 초과근무자료를 날짜별 캘린더로 보여줍니다.

날짜를 클릭하면 해당일의 초과근무자, 실제 시작시간, 실제 종료시간, 실제 시간합을 확인할 수 있습니다.

### 초과근무 표

전체 초과근무 내역을 표로 볼 수 있습니다.

성명 검색, 승인상태 필터, 00:00 제외/포함 기능을 사용할 수 있습니다.

### 카드자료 선택 업로드

BC카드 승인내역 또는 이용내역 엑셀을 추가하면 날짜별 식사후보 금액을 함께 표시합니다.

카드 사용내역만으로 실제 대상자를 확정할 수는 없으므로, 실제 대상자와 사용 목적은 원인행위나 증빙자료로 함께 확인해야 합니다.

---

## 식사후보 상태 🚦

식사후보 상태는 카드 식사후보 금액과 초근자 수를 기준으로 표시됩니다.

| 상태 | 의미 |
|---|---|
| 기준이내 | 식사후보 금액이 초근자 수 × 1인 기준금액 이내인 경우 |
| 초과의심 | 식사후보 금액이 초근자 수 기준 가능액을 초과하는 경우 |
| 근거없음 | 식사후보 카드 사용은 있으나 해당 날짜의 실제 초과근무자가 없는 경우 |

1인 기준금액은 최대 기준액입니다.  
반드시 전액을 사용해야 한다는 의미는 아닙니다.

예를 들어 초근자가 6명이고 식사후보 금액이 3명분 정도만 있어도, 금액상으로는 기준이내입니다.

---

## 1인 기준금액 ⚙️

기본값은 9,000원입니다.

화면의 `1인 기준금액` 입력칸에서 금액을 바꾸면 매식 가능액, 금액 기준인원, 식사후보 상태 계산이 즉시 다시 반영됩니다.

기관 기준이나 시기별 기준금액이 달라질 수 있으므로, 필요한 경우 직접 수정해서 사용하면 됩니다.

---

## 이름 표시와 이름 가리기 🕶️

NEIS 성명값에 괄호 정보가 포함되어 있어도 화면에는 표시하지 않습니다.

예시는 다음과 같습니다.

| 원본 | 화면 표시 |
|---|---|
| 김이박(880101) | 김이박 |
| 이하은(900101) | 이하은 |

`이름 가리기`를 켜면 캘린더, 상세 보기, 초근표, 복사용 텍스트에 표시되는 이름이 일부 가려집니다.

캘린더에서는 좁은 칸에 맞게 짧게 표시합니다.

| 원래 이름 | 캘린더 표시 |
|---|---|
| 김이박 | 김○ |
| 이하은 | 이○ |
| 김영 | 김○ |

상세 보기, 표, 복사용 텍스트에서는 구분이 가능하도록 조금 더 자세히 표시합니다.

| 원래 이름 | 상세/표 표시 |
|---|---|
| 김이박 | 김○박 |
| 이하은 | 이○은 |
| 김영 | 김○ |

원본 데이터는 바꾸지 않고, 화면 표시와 복사 출력에만 적용합니다.

## 개인정보와 자료 처리 🔒

업로드한 파일은 서버로 전송되지 않고, 현재 브라우저 안에서만 분석됩니다.

NEIS 초과근무자료와 카드 이용내역은 저장되지 않습니다.

단, 앱 실행을 위해 SheetJS, JSZip 같은 공개 라이브러리를 CDN에서 불러올 수 있습니다.  
파일 내용 자체를 외부로 보내지는 않습니다.

## 카드내역 기준 💳

카드 파일은 엑셀 헤더를 보고 자동 판별합니다.

### 승인내역조회

- 비교 날짜: 승인일자
- 금액: 승인금액
- 주요 열: 승인일자, 승인시간, 승인번호, 가맹점명/국가명, 승인금액, 매출종류

승인내역에는 취소 건이 함께 들어올 수 있습니다.

같은 승인번호의 승인과 취소는 합산하여 순액 기준으로 반영합니다.

예시는 다음과 같습니다.

| 승인 | 취소 | 반영 |
|---:|---:|---:|
| 30,900원 | -30,900원 | 0원, 식사후보 제외 |
| 90,000원 | -30,000원 | 60,000원 반영 |

### 이용내역조회

- 비교 날짜: 접수일자/(승인일자)의 괄호 안 승인일자
- 금액: 청구금액
- 주요 열: 접수일자/(승인일자), 청구일자, 승인번호, 가맹점명/국가명, 청구금액, 비고

예를 들어 `2026.06.30(2026.06.29)`로 표시된 경우, 실제 사용일은 2026.06.29로 보고 초과근무 날짜와 맞춥니다.

## 주의할 점 📝

식사후보는 가맹점명과 금액 패턴을 기준으로 자동 추정합니다.

카드 이용내역만으로 실제 누가 식사했는지까지 확정할 수는 없습니다.  
실제 대상자와 사용 목적은 원인행위, 품의, 증빙자료, 기관 기준 등을 함께 확인해야 합니다.

이 도구는 특근매식비 검토를 쉽게 하기 위한 확인 보조 도구입니다.  
최종 판단은 업무담당자가 관련 자료를 함께 확인하여 처리해야 합니다.
