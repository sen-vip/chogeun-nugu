# 초근 누구 v0.3.1

NEIS 초과근무자료를 캘린더와 표로 정리해, 특근매식비 검토를 쉽게 도와주는 확인 보조 도구입니다.
BC카드 이용내역 엑셀은 선택으로 추가할 수 있으며, 날짜별 식사 후보 금액과 초근자 수를 함께 보여줍니다.

## v0.3.1 기능

- NEIS 초과근무 최종자료 엑셀 업로드
- 날짜별 초과근무자 캘린더 표시
- 날짜 클릭 시 초과근무자 상세 보기
- 실제 00:00 제외/포함 토글
- 승인상태 필터, 성명 검색
- 매식 가능액 자동 계산
- BC카드 이용내역 엑셀 또는 엑셀이 들어있는 ZIP 선택 업로드
- 카드 승인일자 기준 날짜별 식사 후보 결제 표시
- 카드 식사 후보금액과 초근자 수를 비교해 기준이내/초과의심/근거없음 표시
- 카드 전체 보기 토글
- 초과근무 표, 카드 표, 복사용 텍스트 제공
- 접이식 사용방법/자료 받는 법/개인정보 안내
- 1인 기준금액 수정 기능
- 이름 가리기 기능
- 식사후보 상태 미니 범례

## 사용 방법

1. NEIS 초과근무자료를 업로드합니다.
2. 날짜별 초과근무자를 캘린더에서 확인합니다.
3. 필요 시 BC카드 이용내역을 추가합니다.
4. 식사후보 금액과 초근자 수를 함께 확인합니다.

## 자료 받는 법

### NEIS 초과근무자료

나이스 접속 → 초과근무 확인목록 → 해당 월 조회 → 엑셀 다운로드

### BC카드 이용내역

BC카드 이용내역 조회 → 해당 월 전체 이용내역 조회 → 엑셀 다운로드

기관별 메뉴명은 다를 수 있습니다. 실제 사용하는 정확한 경로가 확인되면 화면 도움말 문구만 교체하면 됩니다.

## 1인 기준금액

기본값은 9,000원입니다.
화면의 `1인 기준금액` 입력칸에서 금액을 바꾸면 매식 가능액, 금액 기준인원, 기준이내/초과의심 계산이 즉시 다시 반영됩니다. 1인 기준금액은 최대 기준액이며, 반드시 전액을 사용해야 한다는 의미는 아닙니다.

## 이름 가리기

`이름 가리기`를 켜면 캘린더, 상세 보기, 초근표, 복사용 텍스트에 표시되는 이름이 일부 가려집니다.

- 3글자 이상: 김민지 → 김*지
- 2글자: 김영 → 김*
- 1글자: *

원본 데이터는 바꾸지 않고 화면 표시와 복사 출력에만 적용합니다.


## 식사후보 상태

- 기준이내: 식사후보 금액이 `실제 초근자 수 × 1인 기준금액` 이내인 경우
- 초과의심: 식사후보 금액이 초근자 수 기준 가능액을 초과하는 경우
- 근거없음: 식사후보 카드 사용은 있으나 해당 날짜의 실제 초과근무자가 없는 경우

식사후보 상태는 금액과 초근자 수를 기준으로 한 확인 보조 표시입니다. 실제 대상자와 사용 목적은 원인행위/증빙자료로 확인해야 합니다.

## 카드내역 기준

BC카드 이용내역 엑셀의 다음 열을 자동 인식합니다.

- 접수일자/(승인일자)
- 카드번호
- 승인번호
- 가맹점명/국가명(도시명)
- 청구금액
- 매출종류
- 비고

날짜 비교는 괄호 안의 승인일자를 기준으로 합니다.
PDF는 화면 출력용 성격이 강하고 금액 열 구조가 엑셀보다 불안정하여 v0.3.1에서는 엑셀 또는 ZIP 사용을 권장합니다.

## 개인정보/자료 처리

업로드한 파일은 서버로 전송되지 않고, 현재 브라우저 안에서만 분석됩니다.
NEIS 초과근무자료와 카드 이용내역은 저장되지 않습니다.

단, 앱 실행을 위해 SheetJS, JSZip 같은 공개 라이브러리를 CDN에서 불러올 수 있습니다. 파일 내용 자체를 외부로 보내지는 않습니다.

## 주의

식사 후보는 가맹점명과 금액 패턴으로 자동 추정합니다.
1인 기준금액은 최대 기준액이며, 반드시 전액을 사용해야 한다는 의미는 아닙니다.
이 도구는 최종 판단 도구가 아니라 확인 보조 도구입니다.
최종 특근매식비 인정 여부는 원인행위, 품의, 실제 사용 목적, 기관 기준 등 업무담당자가 함께 확인해야 합니다.
